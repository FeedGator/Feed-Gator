Feed-Gator by FeedGator

Feed aggregator for Joomla that stores feed data as an article in the Joomla database

Feed aggregator for Joomla that stores feed data as an article in the Joomla database. Feed Gator also support automatic imports with Cron, tag extraction, HTML cleaning and more.

Dependencies

Uses the SimplePie parser.

Install

Use the Joomla installer to install the Feed Gator plugin.
